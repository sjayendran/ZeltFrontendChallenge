# Zelt - Frontend Test

## Objective
- Using the available mock data, create a configurable widget system, with widget list on page load.
- Examples of widgets using existing mock API data:
  - Upcoming Birthdays
  - Upcoming Work anniversaries
  - Pending Absence Requests
  - Missing Documents


## Getting Started

```
npm i
npm run dev
```

### Prerequisites

- NodeJS v16.10
- npm v7.24.0
